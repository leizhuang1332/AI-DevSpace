候选 C · 默认 Tab + 可切并排 · 交互说明: (我推荐)
· 默认 Tab 模式: 简洁,主区空间最大,深度编辑最舒适
· 点击 "并排" 视图切换 → 切换到 3 列 grid(类似 B 但简化,无对比列)
· 点击 "Tab" 视图切换 → 回到 Tab 模式
· 点击 会话 Tab(架构/数据/接口) → 在 Tab 模式下切换当前会话
· 在并排模式下点击某列 → 切到 Tab 模式并激活该会话
· 准入仪表板 5 维度 共享(不分子会话)

该形态优点: 默认简洁 + 高级对比可选;复杂度伸缩灵活;视觉与 ZoneBar 一致
该形态缺点: 状态记忆(Tab vs grid)需持久化;实现略复杂(A+B 之和)
